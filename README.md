下列关于云计算的部署模式中说法不正确的是? 

` `A 专有云:也称私有云，用户自行管理和维护难度低于公有云 -物理机/裸金属的I/P/SaaS都需自己维护，难度高

` `B 公有云:规模化、运维可靠、高扩展性

` `C 混合云:同时兼具专有云隐私性和公有云的扩展性优势

` `D 从运维角度来说，混合云的使用和管理复杂性要高于公有云和专有云。

正确答案：A

公有云的底层都是公有云的厂商来运维，用户只需要运维上层的资源，难度大大降低。
